## **Design Notes**

**Typography:**

* Headlines: Bold, large (48-72pt)  
* Body: Clean sans-serif, readable (24-32pt)  
* Minimal text per slide ‚Äî you're the content, slides are support

**Colors:**

* Consider Fractions Studio brand colors if using them  
* Dark backgrounds for emphasis slides (quotes, key points)  
* Light backgrounds for framework/table slides

**Visuals:**

* Icons over bullet points where possible  
* Tables for comparisons  
* Diagrams for loops/frameworks  
* Minimal decoration ‚Äî content first

**Transitions:**

* Simple fades or cuts  
* No flashy animations ‚Äî they distract  
* Pause on key slides, move quickly through support slides
