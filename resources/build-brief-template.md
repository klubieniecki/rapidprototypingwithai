# Build Brief Template

A 2-5 page requirements doc designed for AI coding agents. Not a PRD. Not 50 pages. Just enough context for AI to build something real.

---

---

## 2. Problem

> Quantified pain, trigger moments, workarounds.

**The Pain:**
- What specific problem are they experiencing?
- How often does it happen?
- What does it cost them? (time, money, frustration)

**Trigger Moments:**
- When does this problem become urgent?
- What event makes them search for a solution?

**Current Workarounds:**
- How do they solve this today?
- Why is that solution insufficient?
